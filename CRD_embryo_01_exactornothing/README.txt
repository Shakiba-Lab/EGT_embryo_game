
The code for replicator dynamics CDR game is found in
CRD_embryo_replicator_202501.nb 

The code for stream plots for all the figures in the paper is found in
CRD_streamplot-2026.nb

Demo of the simulations and the data for all the figures that generate the pie charts are found in the folder 2026_replicator_simulations

all files use Mathematica, the simulations were conducted on Versions >11

A single iteration in the stream plot takes <2 mins
A single condition in replicator sims takes <5 mins, and 20000 initial conditions ~3 days, depending on the round number and strategy space
